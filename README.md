# ZhConversion

转换规则基于[维基百科简繁转换规则](https://zh.wikipedia.org/wiki/Wikipedia:%E7%B9%81%E7%AE%80%E5%A4%84%E7%90%86)生成，能正确处理长词组。

----------

## 使用方式

安装到 Greasemonkey 或者 Tampermonkey 等用户脚本插件当中之后，编辑本脚本的自定义包含列表，加上想要启用转换的页面 URL 即可。
